Features

Open Scouting is an open source application for easier scouting at FIRST Robotics competitions

  • Gather pit scouting data
  • Anyone can contribute data, and anyone can view it, no account needed
  • Works offline in case of poor connection in the stands
  • Uses The Blue Alliance to autofill data for faster scouting
  • Create custom events if an event isn't in TBA
  • Manually go offline in case of bad connection
  • Demo mode to let scouts try out the app
  • Share links to an event to start scouting as fast as possible
  • Create an account to verify your scouting reports
